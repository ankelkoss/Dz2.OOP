# csharp-dz2-oop

# Опис
.NET 8 консольний застосунок з об’єктно-орієнтованою архітектурою: для реалізації ДЗ на теми "Гроші та товар", "Музичні інструменти", "Десяткове число" у вигляді консольного меню з можливістю вибору.

# Вимоги
SDK .NET 8 або новіший: https://dotnet.microsoft.com/download
ОС: Windows / Linux / macOS
Бажано: Git 2.4+ для роботи з гілками та PR

# Швидкий старт
dotnet build

# Запустити консольний застосунок
dotnet run --project ./MoneyAndGoods
